Private PDF Deck User Guide

Private PDF Deck helps you organize PDF and image pages locally on your Mac.

1. Add files

Drag PDF, JPEG, PNG, or HEIC files into the window, or use the Add button. Added pages appear in the thumbnail grid.

2. Organize pages

Drag thumbnails to reorder pages. Select pages to rotate, delete, duplicate, crop, or change page size.

3. Use file breaks

Add file breaks when you want to split the export into multiple files. Each section can be exported separately.

4. Check pages in Viewer Mode

Turn on Viewer Mode to inspect the document while keeping the thumbnail grid visible. Use the zoom controls at the bottom right of the viewer.

5. Export

Use Export to save your deck as PDF, JPEG, or PNG. PDF export can include password protection and optional OCR for image pages.

Privacy

Private PDF Deck processes selected files locally. Documents are not uploaded by the app.
